# Lab02 — STP and RSTP

**Goal:**  
Implement and verify Spanning Tree Protocol (STP) and Rapid STP (RSTP) operation between switches.

**Devices:**  
- D1 — L3 Switch (Root Bridge)  
- D2 — L3 Switch (Secondary Bridge)  
- A1 — L2 Switch  

**Files:**  
- `STP.yaml` — CML topology  
- `D1_config.txt`  
- `D2_config.txt`  
- `A1_config.txt`  
- `1.png` — Topology image

**Verification:**  
- STP root bridge election  
- Port roles and states  
- Convergence after link failure  
- Rapid-PVST transition
